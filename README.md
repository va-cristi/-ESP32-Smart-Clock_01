# ESP32 Smart Clock

This project is a smart digital clock built with ESP32 and a 3.2" TFT display.

## Features
- Real-time seconds display
- Scrolling text

## Components
- ESP32 Development Board
- 3.2" TFT SPI Display (320x240)

## Usage
1. Upload the code to ESP32
2. Connect the display
3. Enjoy your smart clock!
